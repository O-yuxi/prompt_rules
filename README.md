# Prompt Rules 提示词规则集

这个仓库包含了各种场景下的提示词规则和指南，帮助用户更有效地与AI助手进行交互。

## 文件说明

- **pm_code.md**: 产品经理与代码开发相关的提示词规则
- **pm_prd.md**: 产品经理与产品需求文档相关的提示词规则
- **java.md**: Java开发相关的提示词规则
- **web.md**: 网页开发相关的提示词规则

## 使用方法

1. 根据您的需求选择相应的提示词规则文件
2. 将文件中的内容复制到您与AI助手的对话中
3. 按照规则与AI助手进行交互
